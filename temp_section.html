08. BUILD PROCESS

The Tech Stack

  • HTML5 Semantic Structuring
  • Vanilla CSS (Grid/Flexbox/Animations)
  • Vanilla JavaScript (ES6+)
  • Intersection Observer API

Implementation Details

This portfolio was built entirely from scratch without complex frameworks to ensure maximum performance and granular control over animations.

  • • Navigation & Scroll: Custom scrollspy logic tracks viewports and updates the sticky sidebar active states in real-time.
  • • Interactive Cursor: A custom JS-driven cursor with lag-trailing effects utilizing requestAnimationFrame for smooth 60fps rendering.
  • • Aesthetic Effects: CSS mix-blend-modes handle the background image integration, and CSS Keyframes drive the title's glitch animation.
